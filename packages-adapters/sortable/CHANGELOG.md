# 1.15.0 (2025-01-24)

### Features

- **sortable:** Initial release of Aurelia 2 wrapper for SortableJS library
- **sortable:** Add support for all SortableJS configuration options
- **sortable:** Add bindable properties for all options (group, sort, delay, animation, etc.)
- **sortable:** Add support for event callbacks (onStart, onEnd, onAdd, onUpdate, etc.)
- **sortable:** Add reactive property change support via propertyChanged
- **sortable:** Add comprehensive test coverage with Jest
- **sortable:** Add complete documentation with usage examples
